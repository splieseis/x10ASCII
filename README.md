## Installation

```bash
npm i x10ASCII
```

## Usage

```javascript
const { x10 } = require('x10ASCII');

// prints out available chars
console.log(x10("0123456789!+-\nABCDEFGHIJKLM\nNOPQRSTUVWXY\nZ.(_)$\"@\"%\'?\'"));
```
